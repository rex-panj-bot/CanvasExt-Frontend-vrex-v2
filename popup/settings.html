Google Gemini API Key (Required)

Use your own Google Gemini API key to access the study assistant with Google's free tier.

API Key:

Quick Setup Guide (2 minutes):
  1. Go to Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Select "Create API key in new project" or choose an existing project
  5. Copy the API key (starts with "AIza...")
  6. Paste it in the box above and click "Save API Key"

Google Gemini Free Tier Limits:

  • 15 requests per minute (RPM)
  • 1 million tokens per minute (TPM)
  • 1,500 requests per day (RPD)
  • No credit card required

Privacy: Your Google API key is stored locally in your browser and only sent to Google's Gemini API (never to our servers).
